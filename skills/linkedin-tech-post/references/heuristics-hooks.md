# Hooks de Alto Sinal

Arquivo de referência com hooks que funcionaram ou que seguem padrões de alto impacto. Atualizar conforme novos posts são publicados e analisados.

## Padrões de hook que geram comentário

### 1. Confissão técnica
O autor admite algo inesperado — gera identificação e respostas do tipo "aconteceu comigo também".

```
"Gastei 3 semanas otimizando uma query. A solução foi deletar ela."
"O melhor código que escrevi esse ano foi o que convenci o time a não escrever."
```

### 2. Contradição com expectativa
O resultado contraria o que a maioria esperaria — gera curiosidade e debate.

```
"Copilot não substituiu nenhum dev no meu time. Mas mudou como todos trabalham."
"Implementei agentes de IA em produção. A parte difícil não foi a IA."
```

### 3. Padrão revelado
Uma observação que conecta vários casos a uma causa-raiz — gera confirmação ou discordância.

```
"90% dos incidentes no meu time tinham a mesma causa-raiz. Não era técnica."
```

### 4. Problema que ninguém resolve
Uma dor organizacional que todos reconhecem mas ninguém enfrenta — gera desabafo compartilhado.

```
"O Azure Advisor sabe exatamente o que seu ambiente precisa. Mas muitas vezes, ninguém aplica."
"Todo mundo sabe que roda container como root — ninguém quer ser o primeiro a mudar."
```

### 5. Antes/depois com contraste forte
Números que mostram uma mudança significativa — gera perguntas sobre "como?".

```
"Antes: 22 recomendações paradas. Depois: 9 resolvidas em 30 minutos."
```

## Anti-padrões (hooks que morrem)

```
❌ "Olá rede! Hoje quero compartilhar..."
❌ "É com grande satisfação que anuncio..."
❌ "5 dicas para melhorar seu [X]" (sem tensão)
❌ "A IA vai mudar tudo" (abstrato demais)
❌ "Concordam? 🚀🔥💡" (engagement bait)
```
