# Comment Patterns

Diretório para armazenar padrões de comentários que indicam alto engajamento.

## Tipos de comentários a capturar

### 1. Identificação
O leitor se vê na situação descrita.
```
"this is such a real problem"
"isso acontece toda semana no meu time"
"finally someone said it"
```

### 2. Experiência complementar
O leitor adiciona seu próprio caso.
```
"in my experience, the biggest issue was..."
"no nosso caso, o que funcionou foi..."
"we had a similar situation where..."
```

### 3. Discordância construtiva
O leitor discorda e explica por quê.
```
"I see it differently because..."
"o contra-argumento aqui seria..."
"this works if X, but in our case..."
```

### 4. Pergunta de aprofundamento
O leitor quer saber mais.
```
"how did you handle the edge case of...?"
"como ficou o rollback plan?"
"what about environments where...?"
```

## Como usar

- Após cada publicação, salvar os melhores comentários aqui
- Usar como referência para calibrar hooks e CTAs futuros
- Comentários de "identificação" indicam que o hook + ângulo funcionaram
- Comentários de "experiência complementar" indicam que o CTA funcionou
