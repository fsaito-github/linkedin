# Post Package — Schema de Saída Final

O agente deve entregar **todos** os itens abaixo ao apresentar um post ao autor.

## Saída obrigatória

```
═══════════════════════════════════════════
𝗣𝗔𝗖𝗢𝗧𝗘 𝗗𝗘 𝗣𝗨𝗕𝗟𝗜𝗖𝗔𝗖̧𝗔̃𝗢
═══════════════════════════════════════════

𝗣𝗼𝘀𝘁 𝗣𝗿𝗶𝗻𝗰𝗶𝗽𝗮𝗹 (PT-BR)
---
[texto completo do post]
---
Caracteres: [N]/2950

𝗣𝗼𝘀𝘁 𝗣𝗿𝗶𝗻𝗰𝗶𝗽𝗮𝗹 (EN)
---
[texto completo do post]
---
Caracteres: [N]/2950

𝗛𝗼𝗼𝗸 𝗔𝗹𝘁𝗲𝗿𝗻𝗮𝘁𝗶𝘃𝗼
→ "[hook alternativo para o autor escolher]"

𝗖𝗧𝗔 𝗔𝗹𝘁𝗲𝗿𝗻𝗮𝘁𝗶𝘃𝗼
→ "[CTA alternativo]"

𝗙𝗼𝗿𝗺𝗮𝘁𝗼
→ [texto / carrossel / texto + banner]
→ Justificativa: [por que este formato]

𝗦𝗰𝗼𝗿𝗲𝗰𝗮𝗿𝗱
→ Comentabilidade: [X]/6
→ Substância: [X]/5
→ Veredicto: [APROVADO / MELHORAR / REESCREVER]

𝗔𝗿𝗾𝘂𝗶𝘃𝗼𝘀 𝗦𝗮𝗹𝘃𝗼𝘀
→ [path do arquivo PT-BR]
→ [path do arquivo EN]
→ [path do banner/carrossel, se aplicável]

𝗣𝗹𝗮𝗻𝗼 𝗱𝗲 𝗣𝘂𝗯𝗹𝗶𝗰𝗮𝗰̧𝗮̃𝗼
→ Melhor horário: [dia e hora sugeridos]
→ Comentário inicial sugerido: "[pergunta de follow-up para a golden hour]"
→ Pessoas para considerar marcar: [1-3 nomes]

𝗗𝗶𝗰𝗮 𝗱𝗲 𝗚𝗼𝗹𝗱𝗲𝗻 𝗛𝗼𝘂𝗿
→ Fique disponível por 60 min após publicar
→ Responda todo comentário nos primeiros 30 min
→ Não edite o post nas primeiras 2h
═══════════════════════════════════════════
```

## Checklist interno (antes de entregar)

- [ ] Post PT-BR <= 2950 caracteres
- [ ] Post EN <= 2950 caracteres
- [ ] Hook com conflito/tensão (não explicação)
- [ ] Pelo menos 1 dado concreto
- [ ] Episódio real presente
- [ ] CTA respondível e específico
- [ ] Sem links externos no corpo
- [ ] Sem auto-promoção vazia
- [ ] 3-5 hashtags relevantes
- [ ] Score comentabilidade >= 4/6
- [ ] Score substância >= 3/5
