# Winning Posts

Diretório para armazenar posts que tiveram alta performance, com análise do que funcionou.

## Como registrar um post vencedor

Criar um arquivo por post com o formato:

```
post-{N}-{tema}-analysis.md
```

## Conteúdo do arquivo

1. Texto completo do post publicado
2. Métricas de performance (impressões, comentários, taxa de engajamento)
3. Comentários qualificados recebidos (os melhores 3-5)
4. Análise do que funcionou (hook, ângulo, CTA, formato)
5. Heurísticas extraídas

## Critérios para "vencedor"

Um post é considerado vencedor quando atinge pelo menos 2 dos 3 critérios:

- >= 5 comentários qualificados (não genéricos)
- Taxa de engajamento >= 5%
- >= 3K impressões
